# Cryptography and Network Security Assignment

## Description
This project explains three basic encryption techniques used in cryptography:
- Vigenère Cipher (Substitution)
- Rail Fence Cipher (Transposition)
- RSA Algorithm (Asymmetric Encryption)  

---

## Vigenère Cipher
- Polyalphabetic substitution cipher using a keyword  
- Formula: C = (P + K) mod 26  
- Example: HELLO + KEY → RIJVS  

---

## Rail Fence Cipher
- Transposition cipher using zigzag pattern  
- Letters rearranged, not changed  
- Example: HELLO WORLD → HLOLELWRDLO  

---

## RSA Algorithm
- Uses public & private keys  
- Based on prime numbers  

Example:  
p=3, q=11 → n=33, φ=20  
e=3, d=7  

Encryption: C = 4³ mod 33 = 31  
Decryption: M = 31⁷ mod 33 = 4  

---

## Conclusion
These methods show how data can be securely encrypted using different techniques in cryptography.
